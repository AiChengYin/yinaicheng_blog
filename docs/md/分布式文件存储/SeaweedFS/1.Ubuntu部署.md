# SeaweedFS Docker 部署 (Ubuntu 24.04.3 LTS)

本文将介绍如何在 Ubuntu 24.04.3 LTS 系统上使用 Docker 部署 SeaweedFS。SeaweedFS 是一个简单且高扩展性的分布式文件系统。

## 1. 环境准备

确保系统已安装 Docker 和 Docker Compose。

```bash
# 检查 Docker 版本
docker -v

# 检查 Docker Compose 版本
docker-compose -v
```

如果尚未安装，可执行以下命令安装：

```bash
sudo apt update
sudo apt install -y docker.io docker-compose
sudo systemctl enable --now docker
```

## 2. 创建目录与配置文件

为了方便管理，将 SeaweedFS 的相关数据和配置集中存放在一个目录中。

```bash
# 创建 SeaweedFS 工作目录
mkdir -p /opt/seaweedfs/data/master
mkdir -p /opt/seaweedfs/data/volume
mkdir -p /opt/seaweedfs/data/filer
cd /opt/seaweedfs
```

在 `/opt/seaweedfs` 目录下创建 `docker-compose.yml` 文件：

```yaml
version: '3.9'

services:

  master:
    image: chrislusf/seaweedfs:latest
    container_name: seaweedfs-master
    command: master
    ports:
      - "9333:9333"
    restart: always

  volume:
    image: chrislusf/seweedfs:latest
    container_name: seaweedfs-volume
    command: volume -mserver=master:9333
    ports:
      - "8080:8080"
    depends_on:
      - master
    restart: always

  filer:
    image: chrislusf/seaweedfs:latest
    container_name: seaweedfs-filer
    command: filer -master=master:9333
    ports:
      - "8888:8888"
    depends_on:
      - master
    restart: always
```

**组件说明：**
* **master**：管理集群元数据和 Volume 服务器，默认端口为 `9333`。
* **volume**：实际存储文件数据的节点，默认端口为 `8080`。
* **filer**：提供类似传统文件系统（支持目录结构）的访问接口，默认端口为 `8888`。

## 3. 启动 SeaweedFS

进入目录，运行 Docker Compose 命令启动服务：

```bash
cd /opt/seaweedfs
docker-compose up -d
```

查看容器运行状态：

```bash
docker-compose ps
```

## 4. 验证部署

部署完成后，您可以通过以下方式进行验证：

* **Master 状态页面**: 访问 `http://<您的IP地址>:9333` 可以看到集群拓扑结构和 Volume 节点状态。
* **Filer 状态页面**: 访问 `http://<您的IP地址>:8888` 可以进行文件的可视化上传、下载和目录浏览。

### 简单测试文件上传

通过 curl 使用 Filer 上传文件：

```bash
# 上传文件
curl -F file=@/etc/os-release "http://localhost:8888/test-folder/os-release.txt"

# 下载文件
curl "http://localhost:8888/test-folder/os-release.txt"
```

至此，您已在 Ubuntu 24.04.3 LTS 上成功使用 Docker 部署了 SeaweedFS。
