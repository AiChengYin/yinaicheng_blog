> ftp

在 Linux 中，如果你要开放 FTP 服务的话，一般需要开放以下端口号：

* 20/TCP：FTP 数据传输端口
* 21/TCP：FTP 控制连接端口
* passive mode 下使用的端口范围，可以在 FTP 服务器的配置文件中指定，一般默认是 1024 ~ 65535。  
注意：如果使用的是 SFTP（Secure File Transfer Protocol），则需要开放 22/TCP 端口。
