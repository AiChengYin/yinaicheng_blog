# 台阶跳跃方式统计

```java
import java.util.Scanner;

/**
 * 一天一只顽猴想要从山脚爬到山顶，途中经过一个有n个台阶的阶梯，但是这个猴子有个习惯，每一次只跳1步或3步
 * 试问猴子通过这个阶梯有多少种不同的跳跃方式
 * 输入描述：输入只有一个这个数n，0<n<50，此阶梯有多个台阶
 * 输出描述：有多少种跳跃方式
 * @author: yinaicheng
 */
public class DifferentStairJumpingWayNumber {

    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        /*输入的台阶数量*/
        int n = in.nextInt();
        System.out.println(getJumpWayNumber(n));
        in.close();
    }

    public static int getJumpWayNumber(int stepNumber){
        /*如果是1层台阶只有1次跳跃方式（跳一步跳一次），如果是2层台阶也只有1次跳跃方式（跳一步跳两次）*/
        if(1==stepNumber || 2==stepNumber){
            return 1;
        }
        /*如果是3层台阶，有2次跳跃方式（跳三步跳一次或者跳一步跳三次）*/
        if(3==stepNumber){
            return 2;
        }
        return getJumpWayNumber(stepNumber-1)+getJumpWayNumber(stepNumber-3);
    }

}
```
